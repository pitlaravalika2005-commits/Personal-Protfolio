# Personal Portfolio Website

## Overview

A responsive Personal Portfolio Website developed using HTML, CSS, and JavaScript to showcase skills, projects, resume, and contact information.

## Features

* About Me Section
* Skills Section
* Project Showcase
* GitHub & LinkedIn Links
* Resume Download
* Contact Form
* Dark Mode
* Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Run

1. Open the project folder in VS Code.
2. Open `index.html` using Live Server.
3. View the website in your browser.
